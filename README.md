# opensource-lighthouse

汇总统计各「大厂」的开源团队和项目。

## 愿景

- 帮助诸位了解在开源社区各家大公司的参与情况，提供一个好项目的整合。
- 促进国内的「大厂」在开源上有更多的投入，贡献更多能做到「主流」且「国际化」的项目。

## Roadmap
